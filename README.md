# flashcards-front

React/TypeScript/Vite frontend for the flashcards app.

## Run locally

```bash
npm install
npm run dev
```

App runs on **http://localhost:5173**

> Make sure the server is also running on http://localhost:5001 — or update `src/api.ts` to point to the Render URL.

## Deploy (Vercel)

Vercel auto-deploys on push to `main`. No extra config needed.

## Tech

- React + TypeScript
- Vite
- ESLint + Prettier
